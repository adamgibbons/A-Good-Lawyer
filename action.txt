Technical Overview and Strategy

A GOOD LAWYER will first be reformatted according to IDPF/EPUB 3 standard. 
This is a free and open eBook standard and can be validated using the 
epubcheck application (http://code.google.com/p/epubcheck/).

EPUB files are readable on a broad range of hardware systems, including: 
Android tablets and phones; Barnes & Noble's Nook; Linux tablets; Apple's 
iPhone, iPad, and iPod Touch; Sony Reader. EPUB files are readable across 
a variety of software, including: Aldiko (Android); Google Books; iBooks 
(Apple); Mobipocket.

The Amazon Kindle only accepts a proprietary ebook file format (.azw). 
There is a free/open source application, Kindlegear, that performs 
.epub => .mobi file conversions; .mobi files are only different from 
.azw files in that the latter uses a modified serial scheme. 

Sigil (code.google.com/p/sigil/) is a free/open source .epub file 
editing application that can be used to format plaintext/richtext into 
a validated product.

###################################

COVER IMAGE SPECS

Image size should be 590 pixels wide x 750 pixels high
Image resolution should be 72 pixels per inch (ppi) or higher
Use color images, saved in RGB color space
Image format can be JPEG, GIF, or PNG.

BASH SCRIPTING

==zip the file in .epub format==
cd into parent directory
zip -X0 <name>.epub mimetype
zip -Xur9D <name>.epub *

==validate a well-formed .epub file==
cp <name>.epub into epubcheck directory
java -jar epubcheck-3.0b4.jar <name>.epub
